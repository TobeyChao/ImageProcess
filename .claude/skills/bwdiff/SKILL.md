# 黑白差分去背景（bwdiff）

通过黑底图和白底图的像素差值反算 alpha 通道，实现无模型、轻量级去背景。

## 输入解析

从用户消息中提取以下参数：

| 参数 | 标志 | 必填 | 说明 |
|------|------|------|------|
| 黑底图 | `-b` | ✓ | 黑色背景下的主体图像 |
| 白底图 | `-w` | ✓ | 白色背景下的主体图像 |
| 输出路径 | `-o` | 否 | 默认：黑底图文件名 + `_bwdiff.png` |

若用户未提供输入路径或只提供了一张图，用 **AskUserQuestion** 工具询问。

## 执行步骤

### 1. 运行脚本

从项目根目录执行：

```bash
python .claude/skills/bwdiff/scripts/bw_diff.py -b "<黑底图路径>" -w "<白底图路径>" [-o "<输出路径>"]
```

- 两张图必须同机位、同光照、同分辨率，仅背景颜色不同。
- 脚本输出 `Saved: <路径>` 表示成功。

### 2. 报告结果

**成功**：告知输出文件路径，可附上简要说明（如文件大小、原始分辨率）。

**失败**：展示错误信息，并按以下思路排查：

| 错误特征 | 可能原因 | 建议 |
|----------|----------|------|
| 尺寸不一致 | 两张图分辨率不同 | resize 对齐或重新拍摄 |
| `ModuleNotFoundError` | 依赖未安装 | `pip install pillow numpy` |
| 输入文件不存在 | 路径错误 | 请用户确认路径 |

依赖安装命令：
```bash
pip install pillow numpy
```

---

**注意**：输出固定为 RGBA PNG；算法假设两张图仅背景颜色不同，主体位置和光照需完全一致。
